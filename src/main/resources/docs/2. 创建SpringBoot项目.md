#### 2.1 创建Maven项目

##### 技术勘误文档：Knife4j依赖版本问题修复记录

###### 1. 问题描述
在构建Spring Boot 3.2.6项目时，遇到Knife4j依赖无法下载的错误：
```
Dependency 'com.github.xiaoymin:knife4j-openapi3-jakarta-spring-boot-starter:4.3.0' not found
```

###### 2. 问题分析
- **原配置**：使用Knife4j版本 `4.3.0`
- **Spring Boot版本**：`3.2.6`
- **问题原因**：虽然版本4.3.0理论上兼容Spring Boot 3.x，但在实际构建过程中可能存在Maven仓库同步延迟、特定版本临时不可用或兼容性问题，导致依赖无法正常解析和下载。

###### 3. 解决方案
将Knife4j版本从 `4.3.0` 升级到 `4.4.0`：

###### 修改前：
```xml
<properties>
    <knife4j.version>4.3.0</knife4j.version>
</properties>
```

###### 修改后：
```xml
<properties>
    <knife4j.version>4.4.0</knife4j.version>
</properties>
```

#### 另外在接受langchain4j依赖时，似乎存在网络问题，建议使用国际网络。
